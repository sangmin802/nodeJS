http://opentutorials.org:3000/main?id=HTML&page=12

  • http = protocol
  • opentutorials = 도메인 이름(호스트)
  • 3000 = 포트번호. 한대의 컴퓨터에는 여러개의 서버가 있을 수 있고, 어떠한 서버에 접속해야하는지. 여러개의 서버 중 3000번 포트에 서버가 개설된 것
  • main = path(어떤 파일인지)
  • id=HTML&page=12 = query string. 해당 값을 변경하면 웹서버에 데이터값을 보낼 수 있다. 늘 시작은 ?이며 값과 값의 사이는 &, 값의 이름과 값은 =이 있다.(HTML의 12페이지)

http://loaclhost/?id=HTML

?이후의 query string을 변경함으로서, 사용자의 요청에 따라 사용자게에게 보여줄 파일을 바꿀 수 있다.
